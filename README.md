# Zero Foundry GitHub Defaults

This repository provides organization-wide GitHub defaults for Zero Foundry
repositories.

GitHub applies supported community health files from this repository to Zero
Foundry repositories that do not define their own version of the same file.
Repository-specific files should be used when a project needs narrower
instructions.

## Included Defaults

- Pull request template
